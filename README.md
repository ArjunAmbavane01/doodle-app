## Overview

Doodle is an all-in-one platform designed for real-time collaborative drawing and interactive visual content creation. It features a custom-built drawing engine using the HTML Canvas API, combined with a scalable monorepo architecture and robust build, deployment, and collaboration tools to streamline development and user engagement.

### Why Doodle?

This project enables developers to build and deploy scalable, multi-user drawing applications with ease. The core features include:

## Getting Started

### Installation

Build doodle-app from the source and install dependencies:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/arjunbhavan-dev/doodle-app
   ```

2. **Navigate to the project directory:**
   ```bash
   cd doodle-app
   ```

3. **Install the dependencies:**

   Using **docker:**
   ```bash
   # Build and run with Docker
   docker build -t doodle-app .
   docker run -p 3000:3000 doodle-app
   ```

   Using **npm:**
   ```bash
   npm install
   ```

### Usage

Run the project with:

**Using docker:**
```bash
docker run -it {image_name}
```

**Using npm:**
```bash
npm start
```
